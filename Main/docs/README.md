# Main — Instrukcja użytkownika / User guide

## PL
1. Otwórz stronę `Main/index.html` w przeglądarce.
2. Na ekranie głównym zobaczysz logo Neuroshima dopasowane do maksymalnego dostępnego pola (bez zniekształceń) oraz przycisk **Kalkulator**.
3. Kliknij przycisk **Kalkulator**, aby przejść do modułu pod adresem:
   `https://cutelittlegoat.github.io/Neuroshima/Kalkulator/index.html`.

## EN
1. Open `Main/index.html` in a browser.
2. On the main screen you will see the Neuroshima logo scaled to the largest available area (without distortion) and the **Kalkulator** button.
3. Click **Kalkulator** to open the module at:
   `https://cutelittlegoat.github.io/Neuroshima/Kalkulator/index.html`.
