# Main — Instrukcja użytkownika / User guide

## PL
1. Otwórz stronę `Main/index.html` w przeglądarce.
2. Na ekranie głównym zobaczysz logo Neuroshima, panel startowy oraz tło w klimacie postapo: gradient halftone (kropkowana tekstura) i nieregularne plamy przypominające rdzawą blachę.
3. Kliknij przycisk **Kalkulator**, aby przejść do modułu pod adresem:
   `https://cutelittlegoat.github.io/Neuroshima/Kalkulator/index.html`.

## EN
1. Open `Main/index.html` in a browser.
2. On the main screen you will see the Neuroshima logo, the start panel, and a post-apocalyptic background style: a halftone dot gradient and irregular rust-like metal stains.
3. Click **Kalkulator** to open the module at:
   `https://cutelittlegoat.github.io/Neuroshima/Kalkulator/index.html`.
