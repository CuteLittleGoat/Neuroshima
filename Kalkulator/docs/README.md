# Kalkulator — Instrukcja użytkownika / User guide

## PL
1. Otwórz stronę `Kalkulator/index.html`.
2. Zobaczysz ekran z dużym komunikatem **Strona w budowie**.
3. Ekran informuje, że moduł kalkulatora jest jeszcze przygotowywany.

## EN
1. Open `Kalkulator/index.html`.
2. You will see a large **Strona w budowie** (Page under construction) message.
3. The screen indicates that the calculator module is still in development.
